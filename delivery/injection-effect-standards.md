# 注射/填充类术后效果模拟标准 v1（循证 + 厂商官方）

> 用途：给「AI 术后效果模拟」的 prompt + judge 提供**每部位、每机制**的术后理想视觉标准，使 prompt 更精准、judge 校准对齐真实术后。
> 来源：厂商官方（Allergan/AbbVie Juvéderm IFU、Galderma Restylane、Regen AestheFill、Allergan Botox）+ 同行评审（MD Codes de Maio、各部位共识/综述/RCT）。逐项标证据强度（强/中/弱）。
> 编制：2026-06-01，联网检索 5 路并行研究综合。HA 视觉标准跨品牌通用（乔雅登/弗缦/Restylane 同部位视觉目标一致），品牌仅影响流变学/维持时间，不影响"理想术后长相"。

---

## 0. 总纲（喂 prompt/judge 前必读的三条铁律）

### 0.1 机制三分 × 模拟含义（最重要 —— 决定"术后稳定态"怎么画）

| 维度 | **HA 玻尿酸**（乔雅登/弗缦/Restylane） | **胶原刺激剂 PDLLA/PLLA**（艾维岚 AestheFill / 童颜针 Sculptra / 胶原） | **肉毒**（Botox） |
|---|---|---|---|
| 本质 | 凝胶=本身即体积 | 胶原支架刺激剂（微球被吸收→自体生胶原） | 神经调节，**无体积** |
| 即刻效果 | **即刻有真实体积**（偏肿） | **无即刻体积**（仅载体水肿，数天退后≈没变） | **即刻零变化** |
| 起效/稳定 | 即刻→**2 周稳定**（眼下最慢） | 渐进**数周-数月**（PDLLA 12 周+68%胶原 / PLLA 4-6 月达峰） | 动态纹 1-2 周 / **咬肌瘦脸 ~12 周** |
| 维持 | 6-18 月（可酶溶解·可逆） | 18-36 月（基本不可逆） | 3-6 月 |
| 视觉性质 | **即刻、局部、体积/轮廓**（单点塑形） | **渐进、全局、肤质/紧致/整体饱满**（养肤感、年轻几岁，非单点隆起） | **减弱动态纹/瘦脸**，不改体积/肤质 |
| **模拟含义** | 在治疗区画出"多出来的体积/轮廓" | **绝不画即刻局部隆起**；画**整体紧致/饱满/肤质改善**，对标 3-6 月成熟态 | 动态纹：静态中性脸≈无变化（见 0.2）；咬肌：静态正脸宽度收窄 |

> ⚠️ 引擎口径一句话：**HA → 即刻、局部、体积塑形；胶原刺激剂 → 渐进、全局、养肤型年轻化（对标 3-6 月）；肉毒 → 动态纹软化（静态≈无变化）+ 咬肌收窄。**

### 0.2 正脸 vs 侧脸可见性铁律（喂 prompt 的 gate —— 之前正脸鼻/下巴失败的根因）

| 改变 | 正脸 | 斜/侧脸 | prompt 规则 |
|---|---|---|---|
| 泪沟填平 / 黑眼圈淡化 | 🟢 主诉（侧光可见） | 🟢 | 正脸可推 |
| 卧蚕塑造（笑时显） | 🟢（**微笑正脸最佳**） | 🟡 | 优先用含笑表情态 |
| 印第安纹/中颊沟 | 🟢（侧光斜向阴影） | 🟢（45°最突出） | 正脸可推 |
| 唇（唇珠/丘比特弓/比例）| 🟢 | 🟡 | 正脸可推 |
| 苹果肌高点抬升 | 🟢 | 🟢（Ogee 双S须34°斜看） | 正脸推高点，Ogee 放斜脸 |
| 法令纹变浅 | 🟢 | 🟢 | 正脸可推 |
| 太阳穴填平 | 🟡（上外轮廓） | 🟢 | 正脸轻推 |
| **鼻背直化/驼峰填平** | 🔴 **几乎不可见** | 🟢 主战场 | **正脸禁强推**，最多中线高光对称连续 |
| **鼻尖抬升/旋转** | 🟡 仅清晰度/对称 | 🟢 角度只在侧脸 | 正脸轻提清晰度，旋转放侧脸 |
| **颏前突度** | 🔴 **基本不可见** | 🟢 主战场（E-line） | **正脸禁推前突**，放侧脸；正脸只可轻拉长下庭+勾下颌缘 |
| 动态纹（川字/额/鱼尾） | ❌ 仅做表情时 | ❌ | 见 0.3，静态中性脸不可强推 |
| 咬肌瘦脸 | 🟢 下面部宽度 | 🟢 | 正脸可推（但~12 周才达峰） |

> **一句话**：正脸图能做的=泪沟/卧蚕/印第安纹/唇/苹果肌/法令/下庭拉长/下颌缘/瘦脸宽度；**鼻背、鼻尖旋转、颏前突三大效果全在侧脸，正脸强推=失真**。

### 0.3 时间锚点 + 族裔/过度填充红线

- **时间锚点（模拟用"稳定代表态"）**：HA 即刻偏肿→2 周（眼下/卧蚕/中颊沟最慢，体积比即刻略收敛）；胶原刺激剂即刻≈无变化→对标 3-6 月成熟；肉毒即刻零变化→动态纹 1-2 周（仅表情）/咬肌 ~12 周。
- **族裔（必须）**：E-line/比例等高加索标准对**亚洲面孔须更保守**（亚裔唇位偏 retrusive、偏好小巧；颏前突幅度下调）。
- **FOS（面部过度填充综合征）东亚高发**：苹果肌/中面尤其，宁少勿多（pillow face/花栗鼠颊/飞碟脸是红线）。
- **通用红线**：HA 亲水→过量即浮肿；朝合理区间靠一档（自然改善）非命中理论极值；保身份/不磨皮/不漂白肤色/保真实纹理/保表情。

## 1. HA 玻尿酸填充 —— 各部位标准

### 1.1 泪沟 tear trough（指征产品：Volbella XC，2022 FDA 唯一眼下凹陷指征）
- **do_right**：内眦下斜向外下的凹陷填至**齐平**；眼下三角阴影/黑眼圈视觉减轻（增厚菲薄下睑皮肤遮挡其下肌肉静脉，**非漂白肤色**）；睑-颊交界由断层变**平滑连续过渡**；正/斜脸侧光下沟壑投影消失、不显疲惫凹陷。
- **avoid**：① **Tyndall 蓝灰**（填太浅，最易识别的过矫）② 眼下浮肿/malar edema ③ 填出凸度/假性眼袋(pseudoherniation) ④ sunset eyes/松鼠脸。**铁律：填到齐平即止、不透蓝、不把凹陷变隆起。**
- **guardrail**：深层骨膜上(supraperiosteal)，单侧保守 0.2–0.6mL（上限~1.0mL/侧）；薄皮肤用低黏弹/低亲水小颗粒胶防透蓝；意图 60–80% 矫正（留二次补）。
- **time/visibility**：即刻偏肿→1-2 周消肿、2-4 周稳定（眼下最慢）；正脸侧光即可见。
- **evidence**：FDA Volbella XC 眼下指征（强）；Anido 2021 泪沟共识 JOCD（强）；Restylane-L 眼下 3D 量化 ASJ（中-强）。

### 1.2 卧蚕 charming roll ⚠️（**塑造饱满，方向与泪沟相反**）
- **do_right**：紧贴**下睫毛线正下方**（睑前 pretarsal 区）塑造一条**细窄柔和连续的横向饱满隆起**；**微笑时自然鼓起更明显**（"眼睛在笑"的亲和年轻可爱感），静态低调。理想态可"卧蚕饱满 + 其下泪沟平滑"并存。
- **avoid**：① 腊肠卧蚕/sausage-roll（粗圆僵硬）② **混成眼袋**（卧蚕=睫毛线下窄浅愉悦小鼓包；眼袋=更大更下发暗下垂的疲惫袋，**绝不能做成显老眼袋**）③ 整片浮肿/泡眼 ④ 静态就过度膨出（应"笑时显、静时收"）。
- **guardrail**：睑前深层皮下/肌上、紧贴睫毛线；小颗粒低黏弹软胶；极少量、窄带。精确 mL 无共识（需人工核对）。
- **time/visibility**：即刻偏肿/淤青→1-2 周稳定；**微笑正脸最佳可见**（模拟优先含笑表情态）。
- **evidence**：Hong · Life(Basel) 2025（中）；眼整形医生科普（弱-中）。⚠️ 缺厂商专门指征+RCT，量化需人工核对。

### 1.3 印第安纹 / 中颊沟 mid-cheek groove
- **do_right**：颧弓下方**斜向（内上→外下）**横跨前颊的凹陷沟（沿颧颊韧带）**抚平变浅**；上下颊"断层投影"消失、前颊与睑颊-鼻唇过渡平滑连续；常**间接**带动泪沟变浅、中面饱满年轻。
- **avoid**：① **malar edema 颧部水肿**（最关键，过量/浅层挤压淋巴致时好时坏的肿胀）② 加重颊袋/festoon ③ 表面结节/微笑聚团 ④ 整颊均匀鼓/飞碟脸。
- **guardrail**：**分层**（深层 SOOF/deep malar fat 高黏弹支撑 + 浅层低黏弹精修）；每次 0.1–0.2mL 微量推注暂停观察；少穿刺点+钝针防 malar edema；微笑动态评估无聚团。
- **time/visibility**：即刻偏肿→1-2 周消肿、3-4 周稳定；正脸侧光可见、45° 斜脸最突出。
- **evidence**：Hong · JPRAS Open 2025 综述（中，缺RCT）；Mendelson 中颊解剖 PRS 2002/2008（强）；JCAD malar edema 防治（中）。

### 1.4 鼻背 / 鼻（非手术隆鼻）⚠️ 血管最高危区（失明/坏死风险），模拟保守为铁律
- **do_right（侧脸主战场）**：驼峰上下方少量补量使鼻梁**从鼻根到鼻尖呈平滑直线/微凹女性曲线**（折点抹平非削骨）；鼻尖经鼻棘/鼻小柱垫起**适度上旋**（钝化过锐鼻唇角，制造上旋视觉）；正脸仅**中线一条连续不断裂的高光带**+两侧鼻背美学线对称（**正脸改变极微妙**）。
- **avoid**：**Avatar nose**（驼峰上方填太多、额鼻连成片、鼻根过高→鼻整体变宽变重）；鼻背横向增宽/高光成片非一条线；鼻尖过抬(猪鼻)；polly beak/团块；鼻小柱过填致鼻尖下坠。
- **guardrail**：鼻额角女 115–125°/男 120–130°；鼻唇角女 95–110°/男 90–95°（男勿过旋）；**鼻根用量极少**(mild 0.05–0.1mL，鼻根+鼻背上限~0.3mL)，鼻尖 0.05–0.1mL；贴骨/软骨膜深层近中线；**宁可欠矫(undercorrection)**。单病例总量参考 NOSER 仅 0.5mL（"人均0.8mL"是经验值/弱证据）。
- **time/visibility**：即刻偏肿、30 天复评；🔴 **正脸几乎不可见，侧脸才是主战场**——**正脸 prompt 禁强推鼻梁变直，最多中线高光对称连续**。
- **evidence**：PMC10541163 综述（强）；PMC12084090 NOSER 五步法（强）；血管并发症 PMC4859242（中，安全）。

### 1.5 下巴 / 颏（指征产品：Voluma XC=颏部首个FDA / Volux XC=下颌缘）
- **do_right（侧脸主战场）**：颏**前突度增加**、软组织 pogonion 前移、**下庭拉长**、侧面轮廓接近 Ricketts E-line、颏唇沟保持自然弧度；正脸仅下庭略增长+下颌缘到颏过渡流畅（改善"短下巴/没下巴"）。
- **avoid**：巫婆下巴(过尖前突)/桌山方块(中线堆量成台面)/鹅卵石+peau d'orange(橘皮，mentalis 层次错)/过矫致颏下垂(chin ptosis)/结节/不对称。
- **guardrail**：**E-line 上唇后≈4mm、下唇后≈2mm**（高加索值，**亚洲面孔须下调前突幅度**——实测亚裔偏好更小巧/唇位偏后）；颏中线核心点(MD Code C4)~0.3mL/点，全颏可较鼻多；supraperiosteal 贴骨+回抽。
- **time/visibility**：即刻偏肿→2 周稳定；🔴 **颏前突侧脸最明显，正脸基本不可见**——**正脸 prompt 禁推前突，只可轻拉长下庭+勾下颌缘**。
- **evidence**：FDA Voluma 颏部批准（强）；FDA Volux 下颌（强）；MD Codes PMC8012343（强）；E-line+族裔差异 PubMed 19362272（中-强）。⚠️ 面动脉/颏下动脉，贴骨+回抽；HA 可酶溶解（相对鼻区安全优势）。

### 1.6 丰唇 lips（指征产品：Volbella XC / Restylane Kysse）
- **do_right（正脸为主）**：唇珠(central tubercle)形成、丘比特弓双峰中央嵴清晰**保留并强化**(MD Code Lp2 仅 0.05mL定义中央峰而不堆量)、唇红缘清晰、垂直唇高自然增加、上下唇柔和丰盈；微笑说话自然形变不僵。
- **avoid**：香肠唇/鸭嘴(上唇异常前突)/trout pout/球状僵硬；唇红缘被填平消失；口角下垂或反向堆量；人中堆量中央过满；真性鸭嘴(愈合后仍前突=过量+错位)。
- **guardrail**：**上下唇比 1:1 ~ 1:1.6 区间**（同行评审 N=307：1:1 最受偏好、1:1.6 仅年长群体、1:2 显著"不自然像鸭/香肠"；**无单一普适值，勿硬套黄金比 1.618**，族裔差异显著）；丘比特弓中央峰强化非填平；定向焦点丰盈非整唇弥散。
- **time/visibility**：即刻偏肿（24h内常见）→2 周稳定；正脸高度可见。
- **evidence**：Volbella FDA DFU（强）；Restylane Kysse Galderma FDA（强）；MD Codes Lp1-8（强）；PMFA 唇比例研究 N=307（中，注:非题述N=570）。

### 1.7 苹果肌 / 面颊 cheek/malar（指征产品：Voluma XC / Restylane Lyft）
- **do_right（正脸+斜脸）**：颧高点(malar apex)在**外上象限**抬升（**Hinderer 线交点**：外眦下~15-20mm、外侧~10mm 的颧突最大投影点），光反射保持外上象限；斜脸(34°)**Ogee 双S曲线恢复**（颧突凸弧流入 submalar 凹弧）；midface 获支撑抬升、**法令纹间接变浅**（MD Codes 原则：先 Ck1-3 重建中面"地基"再考虑直接处理法令）。女颧高点更宽柔略偏前、男更高锐偏外。
- **avoid**：pillow face/飞碟脸/花栗鼠颊/sunset eyes(吞没睑颊交界)；**submalar 凹弧被填平→失去 Ogee 下半内收→不自然圆鼓**；颧高点错置到鼻侧/弥散铺满；微笑异常前突。**FOS 东亚高发，亚裔尤须保守。**
- **guardrail**：颧高点投影**外上非鼻侧**；**定向投影(骨膜上小bolus)非弥散铺满**；保留 submalar 凹；先建地基再法令；"less is more"。
- **time/visibility**：即刻偏肿→2 周稳定；正脸可见高点，Ogee 须 34° 斜看。
- **evidence**：Voluma XC 中面唯一 FDA（强）；Restylane Lyft FDA（强）；Hinderer线/Ogee 解剖（中-强）；FOS 东亚倾向 PubMed 38369859（强）。

### 1.8 法令纹 nasolabial fold
- **do_right（正脸+斜脸）**：沟变浅但**不完全消失**（保留自然鼻唇过渡）；"间接为主直接为辅"——先苹果肌(Ck1-3)从上方悬吊压缩沟，再 NL1(上段深/骨膜上)/NL2(中段动态)/NL3(下段)针对性精修。
- **avoid**：填到沟**完全消失**(法令纹是正常解剖，全消显假)；过满使鼻旁/颊隆起成嵴；孤立猛填忽略上方苹果肌缺失(MD Codes:孤立填法令不自然)；pillow face 局部。
- **guardrail**：朝目标区间靠一档非全消；优先苹果肌地基（间接），直接填仅剩余精修；深度分层(NL1深/NL2-3皮下)。
- **time/visibility**：即刻偏肿→2 周稳定；正脸高度可见。
- **evidence**：MD Codes NL1-3 + cheek-first 原则 PMC8012343（强）；Voluma "间接减轻法令"（强）；FOS 红线（强-中）。

### 1.9 太阳穴 / 颞部 temple（指征产品：Voluma XC，2024 FDA 颞部指征）⚠️ 血管高危区
- **do_right**：填平颞凹；上面部从骨感/凹陷/憔悴变饱满流畅；**发际-眉尾-颧弓-额头过渡顺滑**（FDA原文"smooth transition from cheekbone to forehead"）；平衡面部比例、改善"太阳穴塌陷显老/显凶"。
- **avoid**：颞区鼓出/膨隆不自然；表浅注射结节/可见隆起；左右不对称。
- **guardrail**：**颞区是面部最高危血管区之一**（颞深动脉↔眼动脉 4 条吻合，误注致失明/卒中/坏死）；FDA Voluma 颞部强制术者培训；骨膜上深层单点+回抽或超声引导；避 tragus-眉弓连线。
- **time/visibility**：即刻 lift；肿胀/结节 2-4 周退；维持 up to 13 月；正脸中（上外轮廓），斜/侧脸更明显。
- **evidence**：FDA Voluma 颞部指征+IFU（强）；颞区血管 meta-analysis（强）。

### 1.10 韧带提升 / 液体提升 liquid facelift（MD Codes 的 lifting/支撑 action，非 filling）
- **do_right（正脸+3/4侧）**：在骨性高位支撑点(颧弓/苹果肌高位/梨状孔/下颌角)注高内聚力 HA 重建"地基"，**向上向外重新支撑中下面部**；下颌缘/法令/木偶纹**间接改善**；轮廓整体上提的"被托起感"——**治因(支撑结构流失)非填果**。
- **avoid**：过量致沉重/pillow face/月亮脸；中面过度外扩变宽变圆；把支撑点当填充点在浅层堆量；追求即刻局部隆起而非整体上提。
- **guardrail**：深层骨膜上固定锚点 + 高 G'/高内聚力产品；少量多点分次评估；MD Codes 每点绑定 fill/support/relax/lift 之一**不可混用动作类型**。
- **time/visibility**：即刻见提升、2 周稳定；正脸高（中面宽度/苹果肌高度/法令/下颌缘）、3/4 侧最明显。
- **evidence**：MD Codes / 8-Point Lift de Maio（中-强，框架同行公认，量化幅度需核对原始培训资料）。

## 2. 胶原刺激剂 PDLLA/PLLA —— 机制标准（艾维岚 AestheFill / 童颜针 Sculptra / 胶原）

**与 HA 的本质区别 = 怎么模拟这类 case 的核心规则：**
1. **HA case 术后态** = 即刻 + 局部 + 体积/轮廓（治疗区画出"多出来的体积"）。
2. **胶原刺激剂 case 术后稳定态 ≠ HA 的即刻局部隆起**，必须表现为：
   - **渐进的整体紧致**（皮肤更紧、松弛改善）
   - **全局饱满**（整脸年轻化、轮廓柔和饱满，非单点鼓起）
   - **肤质改善/养肤感**（弹性、光泽、紧实度提升）
   - **自然、不像"被填过"**——看不出某点多了体积，而是"整张脸状态变好/年轻几岁"
3. **时间线陷阱（最大坑）**：胶原刺激剂**注射后即刻到数天"看起来几乎没变"**（载体退后回基线）。模拟其"术后稳定态"**绝不能画成即刻局部体积爆出**（那是 HA 逻辑会失真），应对标**数周-数月后渐进全局紧致/饱满/肤质提升的成熟效果**（PLLA 4-6 月达峰、PDLLA 12 周+68%胶原类似）。
- **PLLA vs PDLLA**：物性略异（PLLA 半结晶降解慢/PDLLA 无定形多孔降解较快），但**视觉与时间线逻辑一致**（都渐进/全局/养肤），模拟可同类处理。
- **胶原(collagen)填充**：亲水低→即刻膨胀小、无 Tyndall、即刻≈最终态、维持短(3-6月)，模拟即刻态更接近最终态（无大幅消肿过程）。
- **evidence**：PDLLA review PMC11434839（中-强）；PLLA systematic review PMC11435306 + Avelar 2023/2025 Wiley（强）；"12周+68%胶原"来自厂商需人工核对。

## 3. 肉毒 Botox —— 各部位标准（神经调节，无体积）

> **机制警示（贯穿）**：① **注射即刻零变化**；② **动态纹三项在静态中性脸照片上几乎不可见**——纯动态纹肉毒 case 在静态正脸"看起来没变"是**生理正确，judge 不应据此判失败**（须用做表情的对照图验证）；③ 咬肌瘦脸是唯一静态正脸有明显宽度变化者，但**最慢(~12周达峰)**。

### 3.1 咬肌瘦脸 masseter（肌肉麻痹→继发萎缩，非即刻）
- **do_right**：下面部(下颌角/bigonial 宽度)**变窄**、方/国字脸→V/瓜子/卵圆脸轮廓柔化；咬肌膨隆消退、下面部三角收窄；**中面颊饱满度保留**(不连带凹陷)；笑容对称自然。**肉毒唯一静态正脸明显宽度变化。**
- **avoid**：**矛盾性鼓包(PMB)**(咬牙时咬肌前部异常鼓出，皮薄男性多见，最常见特征性畸形)；面颊凹陷/sunken cheek(剂量过大/点位过高，亚洲审美读作疲惫非瘦)；太阳穴下塌；笑容不对称/歪嘴(毒素扩散误伤 risorius/zygomaticus)。
- **guardrail**：只动下面部宽度，**不动中面容积/太阳穴**；笑容对称+咀嚼保留；萎缩约 8-35%(均值~22%)，**非越瘦越好，过度=凹陷**。
- **time/visibility**：**即刻零变化**；麻痹峰值 1-2 周但**可见轮廓峰值~11-12 周**(靠继发萎缩，起效最慢)；维持 3-6 月；正脸高度可见。
- **evidence**：PMC4230655 前瞻(强)；PMC12512921 3D(强)；PMC11938332 颊凹陷/PMB(强)。(咬肌为 off-label 但大量同行评审支撑)

### 3.2 川字纹 / 眉间 glabella（FDA 批准指征）
- **do_right**：**做皱眉表情时**眉间两竖纹变浅变少；静止时眉间更平滑；**保留自然眉动度**(皱眉肌+降眉间肌弱化后额肌相对占优，常带内侧眉轻微自然上提)。
- **avoid**：frozen 僵脸；竖纹100%消失如磨皮；上睑下垂(~0.71%，毒素向下迁移)；眉下垂；皱眉区"打蜡感"。
- **guardrail**：**减弱非消除**，保动度+自然皮肤纹理，不追求"零纹磨皮"。
- **time/visibility**：**即刻零变化**；起效 2-3 天、峰值 1-2 周；维持 3-4 月；**仅皱眉表情时可见，静态中性正脸看不出**。
- **evidence**：BOTOX FDA 指征(强)；RCT 起效/维持(强)；上面部并发症 PMC8328485(强)。

### 3.3 抬头纹 / 额纹 forehead/frontalis（FDA 批准指征）
- **do_right**：**抬眉表情时**额部横纹变浅减少；静止时额头平滑；**保留一定抬眉动度+眉位自然**。
- **avoid**：frozen 僵脑门/横纹100%消失如磨皮("打蜡额头")；**眉下垂(~0.98%，额肌是唯一抬眉肌，过度压制必致眉/睑下垂)**；上睑沉重；**Spock/Mephisto 八字挑眉**(外侧未处理中央被压)；代偿性皱眉。
- **guardrail**：减弱非消除、保抬眉动度；**额肌不可全幅麻痹**；注射点距眶上缘≥2-3cm/距眉≥1.5-2cm 避外侧挑眉。
- **time/visibility**：**即刻零变化**；起效~48h、峰值 1-2 周；维持 3-4 月；**仅抬眉表情时可见，静态中性正脸看不出**。
- **evidence**：BOTOX FDA(强)；PMC8328485 眉/睑下垂率+安全距离(强)；PMC12865869 Spock/frozen(强)。

### 3.4 鱼尾纹 / 眼角 crow's feet（FDA 批准指征）
- **do_right**：**微笑/眯眼表情时**外眼角放射纹变浅减少；静止时眼周平滑；**保留自然笑容+眼周动度**(笑起来眼睛仍自然不僵)。
- **avoid**：frozen 僵眼周/笑容僵硬/纹100%消失如磨皮；笑容假；毒素扩散过低累及颧颊致笑容变形；下睑功能受累。
- **guardrail**：减弱非消除、保自然笑容动度+皮肤纹理，不追求"零鱼尾纹磨皮笑"。
- **time/visibility**：**即刻零变化**；起效 2-3 天、峰值 1-2 周；维持≥4 月(中位120天)；**仅微笑/眯眼时可见，静态中性正脸看不出**。
- **evidence**：BOTOX FDA(强)；两项 III 期 RCT n=1362 PMC5414775(强)；ASJ 综述(强)。

## 4. 给 prompt / judge 的落地规则（直接可编码）

1. **机制分支**（按 brand→project 解析）：HA→即刻局部体积塑形；胶原刺激剂→渐进全局养肤型年轻化(对标3-6月，不画即刻局部隆起)；肉毒→动态纹软化(静态≈无变化)+咬肌收窄。
2. **正脸/侧脸 gate**：正脸图**禁强推**鼻背变直/鼻尖旋转/颏前突（侧脸主战场）；正脸只做该部位正脸可见的改变（见 0.2 表）。
3. **时间锚点**：模拟"术后稳定态"——HA 2周(眼下最慢)、胶原刺激剂3-6月、肉毒动态纹1-2周(仅表情)/咬肌12周；不照搬即刻肿胀。
4. **方向辨析**：泪沟=填平凹陷 / 卧蚕=塑造窄带饱满(笑时显，与泪沟相反) / 印第安纹=抚平斜向沟——judge 按部位分别校验不可混。
5. **族裔/过度**：亚洲面孔 E-line/比例更保守；FOS 东亚高发→苹果肌/中面宁少勿多。
6. **judge 校准对齐真实术后审美**（owner 2026-06-01 标准）：要**自然、保毛孔质感、微微红润健康气色**(真实可信术后感)，不是夸张可辨/磨皮塑料感；纯动态纹肉毒静态无变化不判失败。
